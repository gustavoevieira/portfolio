Detalhes do Projeto: Doceria Doce Sabor

Um site estático com design inovador e cores vibrantes, criado para uma doceria artesanal.

Visão Geral

Este projeto demonstra a criação de um site estático focado em design e experiência visual, simulando a presença online de uma doceria artesanal. O objetivo foi conceber um layout inovador e convidativo, utilizando uma paleta de cores vibrantes inspirada em doces, tipografia elegante e animações sutis para encantar o visitante.

O site apresenta seções como um cabeçalho moderno, área de destaque (hero) com efeito parallax, história da doceria, um menu de delícias, galeria de imagens interativa e um formulário de contato. É um exemplo de como o design front-end pode ser aplicado para criar um impacto visual forte para um nicho específico.

Tecnologias Utilizadas

  • HTML5
  • CSS3
  • JavaScript
