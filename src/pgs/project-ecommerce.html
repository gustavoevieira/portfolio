Detalhes do Projeto: E-commerce

Um sistema completo de de e-commerce desenvolvido com PHP, MySQL, HTML, CSS e JavaScript, incluindo uma área administrativa segura carrinho e checkout.

Visão Geral

Este projeto demonstra a construção de um sistema de e-commerce funcional e seguro, abrangendo desde a interface de compra para o cliente até um painel administrativo robusto para gerenciamento de produtos e pedidos. O objetivo principal foi aplicar e integrar tecnologias fullstack para criar uma solução de loja online completa.

As funcionalidades incluem a exibição de um catálogo de produtos detalhado, um sistema de carrinho de compras interativo, um processo de checkout simplificado, e um painel de controle para administradores realizarem operações CRUD (Criar, Ler, Atualizar, Deletar) em produtos e categorias, além de visualizar pedidos.

Tecnologias Utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • PHP
  • MySQL
